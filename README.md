# 我的世界服务器状态查询插件，适配真寻bot

## 已实现的功能

- [x] 查询服务器昵称
- [x] 查询服务器最大人数,当前人数
- [x] 查询服务器motd
- [x] 查询服务器favicon[无则不发送]
- [x] 返回服务器地址及端口
- [x] 返回服务器在线状态
- [x] 查询服务器延迟

## 未来的功能

- [ ] 获取服务器协议号[没什么用，未开发]
- [ ] 获取服务器官网[如果存在]
- [ ] 获取服务器motd的json版本[已实现但未启用]
