# zhenxun_chafu_Minecraft
我的世界服务器状态查询插件，适配真寻bot
<h1>已实现的功能</h1>
☑️查询服务器昵称\n
☑️查询服务器最大人数,当前人数\n
☑️查询服务器motd\n
☑️查询服务器favicon[无则不发送]\n
☑️返回服务器地址及端口\n
☑️返回服务器在线状态\n
<h1>未来的功能</h1>
🔳查询服务器延时\n
🔳获取服务器官网[如果存在]\n
🔳获取服务器motd的json版本[已实现但未启用]\n
🔳敬请期待\n
