# 我的世界服务器状态查询插件，适配真寻bot
***
## 已实现的功能

- [x] 查询服务器昵称
- [x] 查询服务器最大人数,当前人数
- [x] 查询服务器motd
- [x] 查询服务器favicon[无则不发送]
- [x] 返回服务器地址及端口
- [x] 返回服务器在线状态
- [x] 查询服务器延迟
- [x] 更精确的延迟
- [x] 整理了~~屎山~~代码

## 使用方法
- 安装
  - 将`chafu`文件夹放入`plugins`文件夹或自定义文件夹内
  - 若提示缺少依赖，请执行`pip install ping3`
- 使用
  - 查服 [ip]?[port]
